Приложение task-manager позволяет вести список текущих задач. Для каждой новой задачи есть два поля для заполнения: "тема задачи" и "описание задачи". При необходимости панель для создания новой задачи можно закрыть. При этом введенные данные не сохранятся. Либо панель можно свернуть вниз экрана с сохранением введенных данных. Это позволит позже вернуться к описанию задачи.

Каждая вновь созданная задача попадает в список задач. В списке по каждой задаче выводится содержимое поля "тема задачи" и дата создания задачи. Более подробную информацию по каждой задаче можно посмотреть при клике на задачу. При этом на панели справа отобразится вся имеющаяся информация по ней, а именно "тема задачи", "описание задачи", дата и время создания задачи.

Все задачи можно отфильтровать по дате и "теме задачи". Также можно воспользоваться полем для поиска по "теме задачи". Каждую задачу из списка задач можно удалить при клике по иконке "корзина".
